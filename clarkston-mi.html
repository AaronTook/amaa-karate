AMAA Clarkston, Michigan
Information


Clarkston AMAA's classes are in three sessions (14-15 weeks) each year. All classes are on Tuesday evenings at the Clarkston Community Education building. Little Ninjas classes run 5:30-6:00 P.M. for students ages 4-5. Beginner classes run from 6:10-7:10 P.M. each week. Advanced classes run from 7:20-8:20 P.M. and are the last class of the night.
For more information see the Welcome Packet and our Facebook.
For important notifications, sign up using Remind.
