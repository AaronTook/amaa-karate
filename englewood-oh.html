AMAA Englewood, Ohio
Information



Englewood AMAA's classes are in three sessions (14-15 weeks) each year. All classes are on Thursday evenings at the Earl Heck Community Center. Beginner classes run 5:30-6:30 P.M. each week. Sparing class is held from 6:30-7:00 P.M. for purple belts and above. Advanced classes run from 7:00-8:00 P.M. and are the last class of the night.
For more information see the Flyer and Welcome Packet.
